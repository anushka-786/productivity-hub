# Productivity Hub

A web-based productivity application designed to manage notes and daily tasks efficiently with a clean and user-friendly interface.

## Live Demo
https://anushka-786.github.io/productivity-hub/

## Features

- Create, edit, and delete notes
- Pin important notes for quick access
- Search notes in real-time
- Color-coded notes for better organization
- Categorized notes (Pinned and Others)
- Expandable notes view for full content
- Task manager with completion tracking
- Dark mode / Light mode toggle
- Data persistence using localStorage

## Tech Stack

- HTML
- CSS
- JavaScript

## Key Highlights

- Clean and responsive UI design
- Dynamic DOM manipulation
- Modal-based note editing system
- Efficient state management using localStorage
- Focus on user experience and usability

## Future Improvements

- Export notes functionality
- Cloud-based storage
- User authentication system
